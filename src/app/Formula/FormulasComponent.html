Fórmulas

Para este proyecto hemos usado las siguientes fórmulas:

  1. Amortización Constante (A)
    A = C / n
  2. Interés
    Capital vivo del préstamo * 10% / 100
  3. Cuota Total (Tk)
    Tk = A + Ik
  4. Saldo Restante (Sk)
    Sk = Sk-1 - A

Donde:

  • C: Capital inicial
  • n: Número total de períodos (meses)
  • Sk-1: Saldo del periodo anterior
  • i: Tasa de interés mensual (anual / 12)
